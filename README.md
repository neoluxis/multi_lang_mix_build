# Mix Multi Lang

本仓库是为了学习多种编程语言混合编程。把不同语言的实现分别打包成对应的库，以供其他语言调用，
以此最大地体现不同编程语言的优势

目前学习的语言有

- C
- C++
- C#
- Java
- Rust
- Golang
- Python
- ASM

详细说明可见博客文章
